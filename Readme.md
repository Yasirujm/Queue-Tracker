# Queue Tracker

A web application to manage queues efficiently. Designed for institutions or businesses to automatically assign tokens to users and track queue status in real-time.  

## Features

- **Automatic Token Assignment** – Users get tokens automatically when they join a queue.
- **Queue Status Tracking** – View the current status of all queues.
- **Real-Time Updates** – Front-end reflects changes as they happen.
- **User Management** – Each token is associated with a user account.
- **Queue History** – Track past entries and statuses.

## Tech Stack

- **Backend:** Spring Boot (Java)  
- **Frontend:** React.js 🎨  
- **Database:** PostgreSQL  
- **API:** RESTful endpoints for queue operations
